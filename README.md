# Blog App (Next.js)

## 📌 Project Overview
This is a simple full-stack blog application built using Next.js.  
Users can create, edit, and delete blog posts. The frontend communicates with backend API routes to perform CRUD operations.

---

## 🚀 Features
- Add new blog posts
- Edit existing posts
- Delete posts
- Fetch posts using API
- Dynamic UI updates

---

## 🛠️ Tech Stack
- Next.js
- React (useState, useEffect)
- TypeScript
- API Routes (backend)

---

## 🔄 How it Works
User interacts with UI → Request sent to API → Data processed → Response returned → UI updates
